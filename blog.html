Important F&Q

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is the data representation of the objects
that comprise the structure and content of a document on the web.

2. How do you select an element from the DOM?

To select an element from the DOM I can follow this;
After understanding the DOM as an Object Hierarchy, I can use an ID to Select an Element with .getElementById() or
use Class to Select Elements with .getElementByClass() or Use a CSS Selector to Select an Element with .querySelector().

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at
a higher level in the DOM than the element on which the event originated. It allows us to attach
a single event listener for elements that exist now or in the future.

4. How do you manipulate an element's attributes and styles using the DOM?

Manipulating Element Attributes: If you want to change other attributes like id , href , or src , you can use the setAttribute() method. The setAttribute() method accepts two arguments: The name of the attribute to set. The value of the attribute to set.
